数据库应用程序的核心逻辑是通过 SQL 语句执行的，无论是通过解释器直接发出还是通过 API 在后台提交。 本节中的调优指南有助于加速各种 MySQL 应用程序。 该指南涵盖了读取和写入数据的 SQL 操作、一般 SQL 操作的幕后开销以及在特定场景（如数据库监控）中使用的操作。
